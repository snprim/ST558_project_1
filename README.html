In this vignette, we want to show how to access APIs to retrieve data. We use two NHL repositories as examples: NHL records and NHL stats.

Required Packages

To be able to access data from APIs, you should install and load the httr, jsonlite, and tidyverse packages.

```r
library(httr)
library(jsonlite)
library(tidyverse)
```

baseurl_records <- "https://records.nhl.com/site/api"

getfran <- function(){
  fullurl <- paste0(baseurl_records, "/", "franchise")

  if (expand != ""){
